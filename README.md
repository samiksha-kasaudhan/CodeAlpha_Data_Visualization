# Netflix Data Visualization Project

## Objective
This project focuses on visualizing the Netflix dataset using Python libraries such as Pandas, Matplotlib, and Seaborn. The goal is to identify patterns, trends, and insights through graphical representations.

## Tools & Libraries Used
- Python
- Pandas
- Matplotlib
- Seaborn
- VS Code

## Dataset
Netflix Movies and TV Shows Dataset

## Visualizations

### 1. User Rating Score Distribution (Histogram)
Shows the distribution of user rating scores.

### 2. Top 10 Release Years (Bar Chart)
Displays the years with the highest number of Netflix titles.

### 3. Top 5 Content Ratings (Pie Chart)
Shows the percentage distribution of major content rating categories.

## Key Insights
- Most Netflix titles have high user ratings.
- 2016 contains the highest number of titles in the dataset.
- TV-14 is the most common content rating category.
